A Python-based evaluation framework that tests prompt responses from local Large Language Models.

Features:

* JSON-based prompt test cases
* Automated prompt execution
* Response evaluation
* Token counting
* Latency measurement

The framework runs prompts against locally hosted models using **Ollama** and stores structured evaluation results.
